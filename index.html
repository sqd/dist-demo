How To:

1: F12 to open console.

2: Write your own node class (API) or choose one from the presets. Paste and evaluate the definition in the console.

3: Put the name of the class into field "Node Class". If it turns green, it's valid.

4: Draw your network.

5: Evaluate in console app.nodes[i].start() to call the start() function of your node class on node number i. You can also use other names.

6: Play around!
